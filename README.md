# 🚚 Sistema de Controle de Entregas

Sistema web desenvolvido com **Flask + SQLite** para gerenciamento de entregas, criado com o objetivo de **facilitar a visualização da rotatividade de entregas** em um sistema utilizado no dia a dia.

O projeto surgiu a partir de uma necessidade real: acompanhar de forma organizada as entregas realizadas em um sistema de logística utilizado por familiares, melhorando o controle, a divisão de rotas e a análise das atividades.

---

## 📌 Funcionalidades

* ✅ Cadastro de entregas
* 📋 Visualização das últimas entregas
* 📜 Histórico completo
* ✏️ Edição de registros
* 🗑️ Exclusão de entregas
* 📊 Total de rotas e valores
* 💻 Interface moderna e responsiva

---

## ▶️ Como Executar o Projeto

### 1. Clone o repositório

```
git clone https://github.com/seu-usuario/controle-entregas.git
```

### 2. Acesse a pasta

```
cd controle-entregas
```

### 3. Instale as dependências

```
pip install flask
```

### 4. Execute o projeto

```
python app.py
```

### 5. Acesse no navegador

```
http://127.0.0.1:5000
```
